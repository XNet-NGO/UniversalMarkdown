# UniversalMarkdown

Android Compose markdown renderer with IDE-grade text handling.

## Features

- Native Jetpack Compose rendering (no WebView, no AndroidView)
- Syntax highlighting (37 languages via Prism4j)
- LaTeX math (block + inline via JLatexMath)
- GFM tables, task lists, strikethrough
- RTL/bidi-safe strikethrough with copy-paste support
- Streaming animation for AI chat interfaces
- 60+ LaTeX unicode symbol mappings
- Zero-width character handling for complex scripts
- Text selection across all markdown content
- Link tap + long-press to copy URL
- Code block copy button
- Dark theme defaults
